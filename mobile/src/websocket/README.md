O websocket será utilizado de duas maneiras: 
> 1. Para receber notificações sobre a corrida em tempo indicando modificações no status da corrida. Isto é aplicado tanto para o passageiro quanto para o motorista, para que ambos sejam informados sobre o andamento da corrida.
> 2. Para enviar a localização do motorista em tempo real, para que o passageiro acompanhe no mapa.
